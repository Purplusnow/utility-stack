주사위 굴리기

개수·면 수·보정값을 정해 주사위를 굴리는 무료 온라인 주사위입니다. 굴림 애니메이션 후 합계를 보여주며 보드게임과 TRPG에 적합합니다.

이 도구 소개

보드게임 순서 정하기부터 TRPG의 2d6+3 판정까지 — 주사위 개수, 면 수(2~1000), 보정값을 정하면 주사위들이 데굴데굴 구르다 하나씩 멈추며 합계를 보여줍니다. 물리 주사위처럼 각 주사위는 독립적인 암호학적 난수로 결정됩니다.

사용 방법

  1. 주사위 개수(최대 100)와 면 수(2~1000)를 입력하세요.
  2. 필요하면 합계에 더하거나 뺄 보정값을 입력하세요.
  3. '주사위 굴리기'를 누르면 주사위가 구르다 왼쪽부터 차례로 멈추고 합계가 공개됩니다.

자주 묻는 질문

TRPG 표기법(2d6+3)을 쓸 수 있나요?

개수 2, 면 수 6, 보정값 3처럼 각 칸에 나눠 입력하면 같은 결과를 얻습니다. 결과에는 개별 굴림 값과 합계가 모두 표시됩니다.

주사위 값은 어떻게 정해지나요?

각 주사위는 브라우저의 crypto.getRandomValues 난수로 독립적으로 굴려지며, 화면의 회전 연출은 이미 정해진 값을 보여주는 애니메이션입니다.

면 수가 아주 큰 주사위도 되나요?

2면(동전과 동일)부터 1000면까지 지원합니다. 화면에는 최대 12개의 주사위가 표시되고 나머지는 개수로 요약됩니다.
