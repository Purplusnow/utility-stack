이 도구 소개

점심 메뉴, 당첨자 뽑기, 순서 정하기 — 항목을 입력하면 색깔 세그먼트 휠이 만들어지고, 돌리기 버튼을 누르면 룰렛이 힘차게 돌다가 서서히 감속하며 당첨 칸에 멈춥니다. 당첨은 균등 확률로 결정되고 휠은 정확히 그 칸에 멈추도록 연출됩니다. 항목 세팅을 링크로 공유할 수도 있습니다.

사용 방법

  1. 룰렛 항목 칸에 후보를 한 줄에 하나씩 입력하세요.
  2. 뽑을 개수와 중복 규칙(중복 없음/허용)을 선택하세요.
  3. '룰렛 돌리기'를 누르면 휠이 회전하고 핀이 가리키는 항목이 당첨됩니다.
  4. 여러 개를 뽑으면 첫 당첨은 휠로, 나머지는 결과 목록으로 표시됩니다.

자주 묻는 질문

당첨 확률은 공평한가요?

네. 모든 항목이 같은 확률로 뽑히며, 난수는 브라우저의 crypto.getRandomValues를 사용합니다. 휠 애니메이션은 뽑힌 결과 위치에 정확히 멈추도록 재생됩니다.

항목은 몇 개까지 넣을 수 있나요?

제한 없이 입력할 수 있지만 휠 가독성을 위해 12개 이하를 권장합니다. 최대 100개까지 한 번에 뽑을 수 있습니다.

중복 없이 여러 명을 뽑을 수 있나요?

네. '소진 전까지 중복 없음' 규칙을 선택하면 모든 항목이 한 번씩 뽑힌 뒤에야 반복됩니다.

입력한 항목이 서버에 저장되나요?

아니요. 룰렛은 전부 브라우저 안에서만 동작하며 입력 내용은 전송되지 않습니다.
