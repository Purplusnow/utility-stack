이 도구 소개

커피 내기, 청소 당번, 발표 순서까지 — 참가자와 결과만 입력하면 진짜 사다리를 즉석에서 만들어 애니메이션으로 결과를 보여줍니다. 가로대가 무작위로 배치된 사다리를 각자의 색깔 경로가 타고 내려가며, 결과는 사다리 구조 그대로 정해지기 때문에 조작이 불가능합니다. 링크 공유 버튼으로 우리 팀 명단이 세팅된 사다리를 단톡방에 바로 보낼 수 있습니다.

사용 방법

  1. 참가자 칸에 이름을 한 줄에 한 명씩 입력하세요 (최대 16명).
  2. 결과 칸에 벌칙이나 역할을 한 줄에 하나씩 입력하세요.
  3. '사다리 타기' 버튼을 누르면 사다리가 생성되고 경로 애니메이션이 진행됩니다.
  4. '공유 링크 복사'로 같은 명단이 세팅된 사다리를 팀에게 보낼 수 있습니다.

자주 묻는 질문

결과가 정말 무작위인가요?

네. 가로대 배치는 브라우저의 암호학적 난수(crypto.getRandomValues)로 생성되고, 짝은 실제 사다리 경로를 따라가서 결정됩니다. 미리 정해진 결과를 사다리 모양으로 포장하는 방식이 아닙니다.

몇 명까지 참가할 수 있나요?

최대 16명까지 지원합니다. 결과 항목이 참가자보다 적으면 순환하며 배정됩니다.

입력한 이름이 서버로 전송되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 이름과 결과는 어디에도 저장되거나 전송되지 않습니다.

다시 뽑으면 결과가 바뀌나요?

네. '다시 타기'를 누를 때마다 완전히 새로운 사다리가 생성됩니다.

공유한 링크를 받은 사람에게는 무엇이 보이나요?

내가 입력한 참가자와 결과가 그대로 세팅된 사다리가 열립니다. 받은 사람이 직접 버튼을 눌러 추첨을 진행할 수 있습니다.
