حول هذه الأداة

تساعدك Regex مختبر على إنجاز مهمة المطور بسرعة داخل المتصفح. أدخل القيم، راجع النتيجة، ثم انسخها أو شاركها عند الحاجة.

طريقة الاستخدام

  1. افتح Regex مختبر وراجع المثال الافتراضي.
  2. استبدل القيم التجريبية ببياناتك.
  3. راجع النتيجة المحدثة في مساحة الأداة.
  4. انسخ أو حمّل أو شارك النتيجة عند توفر ذلك.
