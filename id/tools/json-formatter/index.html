Tentang alat ini

JSON Pemformat Inspector membantu menyelesaikan tugas Developer di browser. Masukkan nilai, tinjau hasil, lalu salin atau bagikan jika diperlukan.

Cara menggunakan

  1. Buka JSON Pemformat Inspector dan lihat contoh awal.
  2. Ganti nilai contoh dengan input Anda.
  3. Periksa hasil yang diperbarui di area alat.
  4. Salin, unduh, atau bagikan hasil jika tersedia.

Pertanyaan umum

Apakah gratis?

Ya. JSON Pemformat Inspector gratis digunakan.

Tidak. Alat dapat digunakan tanpa pendaftaran.
