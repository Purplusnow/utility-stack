About this tool

Translate file and storage sizes between bytes, KB, MB, GB, and TB using binary base-1024 math, where 1 KB is 1,024 bytes and each step up multiplies by 1,024 again. This matches how operating systems like Windows report sizes, which is why a "500 GB" drive shows up smaller once formatted.

How to use

  1. Enter the size in the Amount field.
  2. Choose the current unit under From: Bytes, KB, MB, GB, or TB.
  3. Choose the target unit under To and read the converted figure.

Frequently asked questions

Does this converter use 1,000 or 1,024 per step?

1,024. One KB is 1,024 bytes, one MB is 1,048,576 bytes, one GB is 1,073,741,824 bytes, and one TB is 1,099,511,627,776 bytes, the binary convention most operating systems use.

Why does my 1 TB drive show as roughly 931 GB?

Drive makers advertise decimal terabytes (1 trillion bytes), but your OS divides by 1,024 steps. Enter 1,000,000,000,000 bytes here and convert to GB to see the 931.32 GB figure yourself.

Is a KB here the same as a KiB?

Numerically yes. The IEC names the 1,024-based units KiB, MiB, and GiB, while this tool uses the traditional KB, MB, and GB labels for the same binary quantities.

How do I estimate download time from a file size?

Convert the file to MB, then note that internet speeds are quoted in megabits, which are one eighth of a megabyte. A 100 MB file on a 100 Mbps connection takes roughly 8 seconds plus overhead.

How precise are the results?

Up to six decimal places, so converting 1,024 MB to GB shows exactly 1 and odd byte counts produce precise fractional values.
