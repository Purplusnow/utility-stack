About this tool

This resizer changes an image's pixel dimensions directly in your browser and gives you a downloadable file, with no upload step. You can type exact output dimensions or pick a preset such as Full HD, Open Graph, or Story, and a ratio lock keeps the picture from distorting. Results show the before and after dimensions, file sizes, and megapixels.

How to use

  1. Choose a PNG, JPG, WebP, or AVIF file under Image file.
  2. Pick a Preset like Full HD 1920 x 1080, or enter your own Output width and Output height.
  3. Leave Keep original ratio checked to avoid stretching, and set the Format and Quality for the output.
  4. Click the download button to save the resized image.

Frequently asked questions

Is my image uploaded to a server?

No. The image is processed entirely in your browser using the canvas API and never leaves your device. Closing the tab discards everything.

What do the presets do?

Presets fill in common target sizes: Full HD 1920 x 1080, HD 1280 x 720, Square 1080 x 1080, Open Graph 1200 x 630, and Story 1080 x 1920. With Keep original ratio checked, the image is scaled to fit inside the preset box without distortion.

How does Keep original ratio work?

If you enter only a width, the height is calculated automatically, and vice versa. If you enter both while a preset is selected, the image is scaled to fit within those bounds while preserving its proportions.

Which formats can I import and export?

You can open PNG, JPG, WebP, and AVIF files and export to any of those four formats. AVIF export depends on your browser's encoder support; if it is unavailable the tool reports that it could not process the image.

Does the Quality setting affect PNG output?

PNG is lossless, so the Quality value mainly matters for JPG, WebP, and AVIF, where lower values produce smaller files with more compression artifacts. The default is 90.

Does resizing keep EXIF data like GPS location?

No. The image is redrawn on a canvas and re-encoded, which does not carry over EXIF metadata such as camera details or location.
