About this tool

Test a meta description against the length range that fits Google's search snippet. Paste the description to see its character count, the recommended 120-160 window, and an at-a-glance status.

How to use

  1. Paste your snippet text into the Meta description field.
  2. Check the Characters count against the Recommended 120-160 range.
  3. Rewrite until the Status shows Good.

Frequently asked questions

Why does the recommended range start at 120?

Very short descriptions leave snippet space unused and give searchers little reason to click. Aiming for 120-160 characters fills the desktop snippet without being cut off.

Will Google always use my meta description?

No. Google rewrites descriptions for a large share of queries when it thinks page content matches the search better. A well-written description still helps for your primary queries.

Do meta descriptions affect rankings?

Not as a direct ranking factor. Their value is click-through rate: a compelling, complete snippet wins more clicks from the same position.

Is the mobile limit different?

Mobile snippets are often shorter, closer to 120 characters, so front-load the key message in the first 120 characters even when using the full range.

Is my description text kept on my device?

Yes. The counter runs entirely in your browser; the text is never transmitted.
